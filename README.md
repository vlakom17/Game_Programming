# Программирование игр
**Комбаров Владислав, М8О-412Б-22**

---

## Основные механики

1. **Взаимодействие с объектами исключительно через управляемый луч**, имеющий два режима — притяжение и отталкивание. Каждый режим имеет свой ограниченный ресурс энергии. Мощность излучения увеличивается при удержании кнопки мыши.

2. **Физическое взаимодействие с объектами** — расстояние, на которое перемещается объект, зависит от размера объекта и мощности излучения: чем больше объект, тем тяжелее его сдвинуть, и в то же время чем сильнее излучение, тем дальше объект переместится.

---

## Описание игры

Главная цель игрока — пройти уровень, состоящий из нескольких комнат, используя луч для взаимодействия с объектами и окружением.

Игрок должен:
- преодолевать препятствия;
- управлять ограниченным запасом энергии;
- собирать ключи для открытия двери;
- в конце уровня притянуть трофей.

Механика луча включает два режима работы — притяжение и отталкивание объектов.

## Игра включает в себя:

- Игровой уровень с платформами, препятствиями и интерактивными объектами;
- Взаимодействие с объектами в различных физических условиях — объекты могут находиться как на земле, так и в воздухе, что влияет на особенности их перемещения согласно физике Unity;
- Управление персонажем через **WASD**;
- Прыжок через **SPACE**.

### Механика луча

- **LMB** — притяжение объектов;
- **RMB** — отталкивание объектов;
- Физическое взаимодействие с объектами через Rigidbody и Unity Physics;
- Ограниченный запас энергии для каждого режима луча;
- Увеличение силы луча при удержании кнопки мыши;
- Визуализация луча через LineRenderer;
- Подсветка Interactable объектов при наведении курсора.

### Игровые элементы

- Система ключей и двери:
  - сбор ключей;
  - открытие двери после сбора необходимого количества ключей.
- Краткое обучение перед началом игры;
- Трофей, получение которого завершает прохождение уровня.

### Интерфейс

- Полоски энергии;
- Разделение режимов луча по цветам: красный луч притягивает, синий - отталкивает;
- Счетчик ключей;
- Кнопки **Restart** и **Menu** прямо на уровне для удобной навигации;
- Экран победы (кнопки **Restart**, **Menu** и итоговое время прохождения).

### Звуковое сопровождение

- Звуки луча (на каждый режим свой звук);
- Звук подбора ключа;
- Звук открытия двери;
- Звук победы;
- Фоновая музыка.

### Дополнительно

- Главное меню с запуском игры и выходом из приложения;
- Использование Baked Lighting для предварительно рассчитанного освещения сцены;
- Использование Cinemachine для плавного следования камеры.
